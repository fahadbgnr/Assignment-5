Ans: innerHTML" gives you the full structure,

Question-3: What is event delegation in the DOM?

Ans: a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a mechanism in the DOM’s event propagation model where an event that is triggered on a specific element “bubbles up” through its parent elements.
